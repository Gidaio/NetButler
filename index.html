Welcome to NetButler

Hey, it's NetBulter, a webserver written entirely in Lua! Currently, it only relies on LuaSocket, which is nice. Let's take a look-see at some of the features currently implemented!

  • Four MIME types: text/html, image/jpg, image/png, and image/gif!
  • GET requests! (obviously)
  • 404 error handling!
  • Dynamic pages! Different number each time: ||random||
  • Images!

It's also got support for URL-encoded data in GET requests... but no POST support :( ... Yet!
